# hostsblue

Domain registration, WordPress hosting, professional email, SSL certificates, and AI website builder — everything your business needs online.

## Services

- **Domains** — Register, transfer, and manage 700+ TLDs
- **Hosting** — Managed WordPress, cloud hosting, and VPS
- **Email** — Professional email on your domain
- **Security** — SSL certificates and SiteLock malware protection
- **Website Builder** — AI-powered site builder

## Stack

- **Frontend:** React 18, React Router 6, Tailwind CSS, Vite
- **Backend:** Express, Drizzle ORM, PostgreSQL
- **Payments:** swipesblue.com

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
